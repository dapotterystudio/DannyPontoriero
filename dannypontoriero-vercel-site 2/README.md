# Danny Pontoriero Website

This is a plain static HTML/CSS/JavaScript website prepared for GitHub and Vercel deployment.

## Vercel Settings

- Framework preset: Other
- Build command: leave empty
- Output directory: leave empty, or use `.`
- Install command: leave empty

The root folder should contain `index.html`, the CSS/JS files, image assets, `kikisrecipes/`, and `vercel.json`.
